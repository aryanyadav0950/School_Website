# 🎓 School Website

🌐 **Live Website:** https://bpsnidholi.netlify.app

---

## 📌 About the Project  

This is a **modern, responsive, and professional school website** built for **Bright Public School, Nidholi Kalan (Etah, Uttar Pradesh)**.

The website is designed with a **premium UI/UX**, inspired by: 

* International school websites
* DPS-style layouts
* Apple-level clean design

---

## ✨ Features

* 🎯 Fully responsive (Mobile, Tablet, Desktop)
* 🎨 Modern UI with glassmorphism & gradients
* ⚡ Smooth animations and transitions
* 🖼️ Dynamic image gallery (masonry layout)
* 📊 Animated statistics counters
* 📱 Click-to-call & WhatsApp integration
* 📩 Working admission form (Netlify Forms)
* 🛡️ Spam protection (reCAPTCHA + honeypot)
* 🌍 Deployed on Netlify
